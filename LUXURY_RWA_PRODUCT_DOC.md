
## MantleLuxury 最终产品文档（PRD）

**版本：** v1.0  
**最后更新：** 2025-12-01  
**产品类型：** 奢侈品 RWA 代币化投资平台（基于 Mantle L2）

---

## 1. 产品概述

### 1.1 产品定义

**MantleLuxury** 是一个基于 Mantle 网络的奢侈品实物资产代币化平台，将名表、珠宝、限量包等高价值奢侈品以合规、安全的方式拆分为可交易的链上代币，让全球投资者以较低门槛参与奢侈品投资，并通过资产升值与租赁获得收益。

### 1.2 产品愿景

- **成为奢侈品 RWA 领域的基础设施和首选平台**
- 让奢侈品从“少数人的收藏品”变成“多数人的可投资资产”
- 打通「实物资产 → 代币化 → 交易/租赁 → 收益分配 → 合规报告」的完整闭环

### 1.3 目标

- 在 Mantle 生态内，成为 RWA 赛道标杆项目  
- 上线首批 5–10 个试点资产，验证商业和技术可行性  
- 为投资者提供 **透明、安全、流畅** 的奢侈品投资体验  
- 为资产持有者提供 **高效变现与流动性** 解决方案

## 2. 用户与使用场景

### 2.1 用户角色

1. **个人投资者**
   - 画像：有一定理财经验，愿意配置另类资产，金额 1,000–50,000 美元级别
   - 需求：低门槛参与奢侈品投资、提升收益率、可视化资产与收益

2. **机构投资者**
   - 画像：家族办公室、资产管理机构、对冲基金等
   - 需求：合规透明的另类资产标的、API 接入、批量操作能力、报表导出

3. **收藏家 / 奢侈品持有者（资产发行方）**
   - 画像：拥有多件名表、珠宝、包袋，希望盘活资产
   - 需求：在不完全出售实物的前提下获得流动性；保持品牌和资产价值

4. **平台运营与合规人员**
   - 需求：管理资产上架、审核 KYC/AML、发起收益分配、生成合规报告

### 2.2 核心使用场景

1. **碎片化奢侈品投资**
   - 用户用几百美元，即可购买顶级名表/钻石的份额代币
   - 平台展示资产详情、估值报告、真伪证明和历史表现

2. **资产持有者获取流动性**
   - 收藏家将价值 50 万人民币的百达翡丽代币化为 1,000 份
   - 出售部分代币获得现金流，仍保留部分代币分享未来升值

3. **租赁收益共享**
   - 平台将名表/珠宝租赁给活动方、广告拍摄等
   - 租金扣除管理费后，按持币比例通过智能合约自动分配

4. **合规报表与监管对接**
   - 用户导出年度收益、交易记录，用于税务申报
   - 平台生成必要的合规与审计报告，对接合作机构和监管

## 3. 产品范围与原则

### 3.1 第一期产品范围（MVP）

- 支持 **名表、珠宝** 两类资产的代币化与交易
- Web 端应用（桌面优先，兼顾移动浏览器）
- 基础 KYC（个人为主）与黑名单/制裁名单筛查
- 资产展示、代币购买、资产持仓与收益查看
- 单次性 **资产升值收益分配**（资产出售后分配）

### 3.2 非目标范围（MVP 不做）

- 完整 DAO 治理与平台代币经济
- 开放 API 与第三方大量集成
- 完整多地区税务自动上链申报
- 全功能移动 App（原生 iOS/Android）

### 3.3 产品设计原则

- **合规优先**：所有设计默认遵循 KYC/AML、资产证明和监管要求
- **资产安全优先**：托管、保险、多签、审计必须落实
- **体验精致**：视觉和交互需匹配“奢侈品”定位
- **透明可验证**：重要状态均在链上可查，链下有审计与报告

## 4. 关键业务流程

本节从产品视角描述端到端流程，供设计、研发和合规协同使用。

### 4.1 新用户投资流程（个人投资者）

1. **注册 & 连接钱包**
   - 用户访问官网，点击「连接钱包」
   - 支持 MetaMask、WalletConnect 等主流钱包
   - 未完成 KYC 前，仅可浏览资产，无法购买

2. **KYC 实名认证**
   - 填写基本身份信息（姓名、证件号、地址、联系方式）
   - 上传身份证件 + 人脸识别（第三方 KYC 服务）
   - 简单风险测评问卷
   - 审核通过后，链上 `KYCRegistry` 记录用户状态

3. **浏览并选择资产**
   - 在资产列表页按「类别/品牌/价格区间/风险等级」筛选
   - 点击进入资产详情页，查看：高清图片、3D 模型、估值报告、托管/保险信息

4. **购买代币份额**
   - 在资产详情页输入购买金额或份额
   - 系统展示预计持有比例、费用和风险提示
   - 用户在钱包中确认交易，`LuxuryToken` 合约完成代币转账

5. **查看持仓与收益**
   - 在「投资组合」页面查看各资产持仓占比、估值变化
   - 收益分配发生时，`YieldDistribution` 合约自动向用户地址转账

### 4.2 资产上架与代币化流程（发行方）

1. **发行方申请**
   - 发行方创建「发行方账户」，完成机构级 KYC
   - 提交基础资料（公司信息、授权人信息等）

2. **资产信息提交**
   - 在「提交资产」表单中填写品牌、型号、年份、购入价格等
   - 上传高质量图片与视频、购入凭证、保单等

3. **真伪鉴定与估值**
   - 平台与第三方鉴定机构（可结合 AI 图像识别）完成真伪认证
   - 多家估值机构给出估值区间，形成估值报告

4. **托管与保险落地**
   - 资产实物进入合作托管仓储
   - 购买全险，保险金额与估值挂钩
   - 托管与保险信息写入链上 `CustodyManager` 合约

5. **代币发行与上线**
   - 确定代币总量与最小认购金额（如 1,000 份，每份 500 美元）
   - 部署/实例化 `LuxuryToken` 合约，写入 IPFS 元数据哈希
   - 资产在前端「可投资资产」列表中展示，可开启募集

6. **资产出售与收益分配**
   - 资产到期或满足条件后，由多签发起出售流程
   - 出售成交后，收益进入收益池
   - `YieldDistribution` 按代币持有比例分配升值收益

### 4.3 租赁收益流程（后续版本）

> MVP 可先不实现，预留合约与产品交互接口。

1. 平台与活动方/机构签订租赁协议  
2. 租赁完成后，租金入账平台托管账户  
3. 平台确认净收入（扣除保险附加费、运输费、维护费等）  
4. 调用 `YieldDistribution` 合约，按持币比例发放「租赁收益」

## 5. 功能模块说明

### 5.1 前端功能模块

1. **首页**
   - 精选资产轮播（主推资产、热度排序）
   - 平台关键数据（资产总额、用户数、累计分配收益等）
   - 最近一次收益分配记录

2. **资产列表页**
   - 条件筛选：资产类型、品牌、价格区间、状态（募集中/已满额/已退出）
   - 排序：预期收益率、价格、上架时间等
   - 卡片信息：缩略图、品牌型号、当前价格、剩余可购份额

3. **资产详情页**
   - 资产展示：高清图、3D 模型、视频（如有）
   - 关键信息：品牌、型号、年份、序列号（脱敏）、估值报告链接
   - 托管与保险信息：托管机构、仓储位置（模糊）、保险公司与保额
   - 投资模块：当前价格、剩余额度、输入购买份额/金额、确认按钮
   - 风险提示与合规说明

4. **投资组合页**
   - 总览：资产净值、累计收益、收益曲线
   - 资产分布：按类别/品牌的饼图
   - 持仓列表：支持导出 CSV，用于税务与记账

5. **KYC 页面**
   - 分步流程：基本信息 → 证件上传 → 人脸识别 → 风险测评
   - 实时状态提示（审核中/通过/驳回）
   - 驳回原因与重新提交入口

6. **账户与设置**
   - 绑定邮箱（用于通知）
   - 通知偏好（收益分配、重要公告）
   - 法律文件与风险揭示书

### 5.2 后台运营与合规模块

1. **资产审核后台**
   - 资产提交列表与详情查看
   - 审核记录与备注
   - 一键推送至鉴定机构/估值机构（集成或半手动流程）

2. **KYC/AML 管理**
   - 用户 KYC 状态总览
   - 黑名单与高风险用户标记
   - AML 告警记录与处理状态（与第三方服务对接）

3. **收益分配控制台**
   - 待分配收益列表（升值/租赁）
   - 预估每位持有人收益、Gas 成本提示
   - 触发收益分配交易，多签确认后上链执行

4. **报表与导出**
   - 资产级别收益/费用报表
   - 用户级别交易与收益记录
   - 审计和监管需要的导出格式

### 5.3 合约与链上功能抽象（产品视角）

- `LuxuryToken`
  - 表示单个实物奢侈品的份额代币（ERC-20）
  - 需支持：转账限制（仅通过 KYC 的地址可持有）、资产元数据引用

- `KYCRegistry`
  - 维护地址与 KYC 状态的映射
  - 合约级别提供只读接口，供其他合约检查权限

- `YieldDistribution`
  - 按持币比例，将收益池中的资金分配给持有人
  - 支持不同收益类型（升值/租赁），方便前端展示区分

- `CustodyManager`
  - 记录实物资产托管机构、状态、保险信息（哈希+链下详情）
  - 关键操作（解锁/转移）需多签授权

## 6. 合规与风控要求（产品层）

### 6.1 KYC/AML

- 所有可投资用户必须通过 KYC
- 与第三方 AML 服务对接（如 Chainalysis/Elliptic），检测高风险地址
- 大额交易与异常行为需触发风控规则：人工复核或暂时冻结

### 6.2 资产真伪与估值

- 至少两家权威机构参与真伪认证或估值
- 所有认证报告需存证（IPFS + 链下备份）
- 定期（例如每年）复检高价值资产的状态与估值

### 6.3 托管与保险

- 所有上架资产必须进入合格托管机构，个人保管不予上架
- 托管仓库需满足：恒温恒湿、防火防盗监控、访问控制
- 每件资产必须购买全险，保额不得低于估值

### 6.4 法律与税务

- 针对主要目标市场（如香港、新加坡、欧盟等）进行法律评估
- 平台需提供清晰的使用条款、风险揭示以及投资者适当性说明
- 提供标准化的收益与交易报表，方便用户完成本地税务申报

## 7. 技术与平台集成（面向产品的要求）

### 7.1 Mantle 集成要点

- 所有关键合约部署在 Mantle L2 网络
- 充分利用 Mantle 的低 Gas 优势，实现高频收益分配
- 集成 Mantle 官方推荐钱包与桥接方案，简化资产跨链流程

### 7.2 性能与可用性

- 常规页面加载时间：≤ 2 秒（在主流地区）
- 收益分配相关批量交易，需通过批处理与 L2 优化控制成本
- 提供健康监控与告警，对接运维体系

### 7.3 安全要求

- 智能合约需通过第三方安全审计
- 建立漏洞赏金计划，鼓励社区发现问题
- 后台系统访问需多因素认证与权限分级

## 10. 项目亮点

- **Technical Excellence（技术卓越性）**：  
  - 采用 React/Next.js + TypeScript + Solidity + Mantle L2 的现代化技术栈，前后端与合约均基于成熟、活跃的开源生态  
  - 通过 BFF + 微服务架构、链上链下数据分层、事件驱动的索引与风控体系，保证在资产规模和用户量提升时依然具备良好的可扩展性与可维护性  
  - 引入多签托管、第三方审计、异常交易监控等安全机制，使得高净值资产在链上的全生命周期具备银行级安全标准  

- **User Experience（用户体验）**：  
  - 提供 Web2 级的上手路径（邮箱注册 + 游客浏览 + 渐进式引导到钱包与 KYC），降低非加密原生用户的学习成本  
  - 在资产详情与投资流程中，以高清素材、3D 展示、清晰的步骤引导和风险/收益可视化图表，打造符合奢侈品定位的高端数字体验  
  - 跨设备响应式设计确保桌面和移动端浏览器体验一致，使用户可以随时随地查看持仓和收益  

- **Real-world Applicability（真实世界适用性）**：  
  - 聚焦名表、珠宝、限量包等奢侈品赛道，依托成熟的鉴定、托管、保险与估值体系，确保「链上每个代币背后有真实、可核验的实物资产」  
  - 设计面向个人和机构的双边市场能力（批量操作、报表导出、API 接口），方便家族办公室等专业机构直接纳入其资产配置与合规流程  
  - 通过 KYC/AML、托管/保险要求及标准化报表，帮助用户满足当地监管与税务申报，解决现实世界中最关键的合规落地难题  

- **Mantle Integration（Mantle 集成度）**：  
  - 所有关键合约部署在 Mantle L2，核心资金流转与收益分配在 Mantle 执行，充分利用其低 Gas、高吞吐与与以太坊主网共享安全性的特性  
  - 在合约和前端层面预留与 Mantle 生态 DeFi 协议（借贷、DEX、结构化产品）的标准化接口，为 `LuxuryToken` 未来成为 Mantle 上通用抵押物做好准备  
  - 通过对 Mantle 节点健康、RPC 延迟和区块确认的监控，将基础设施层的稳定性直接转化为终端用户顺滑的交互体验  

- **Long-term Ecosystem Potential（长期生态潜力）**：  
  - 以「奢侈品 RWA 标准协议」为中长期目标，逐步开放合约与接口，让更多团队可以在 Mantle 上发行自身的奢侈品资产  
  - 与奢侈品牌门店、拍卖行、收藏俱乐部合作，将链上持仓映射为线下权益（展览、活动邀请等），把 MantleLuxury 打造成奢侈品数字权益基础设施  
  - 通过 DAO 治理与 Mantle 生态激励计划的协同，让社区和生态共同参与资产筛选、费用结构与收益再投资策略，形成可持续的品牌与社区资产
